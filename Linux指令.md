### 删除指令

==rm==：后面直接跟文件即可删除，删除多个文件，就直接把文件名称写上就好了

参数：

1. ```c++
   rm -r 文件夹名 //这个是递归删除目录下的所有内容包括子文件夹
   ```

2. ```c++
   rm -i 文件夹名 //这个删除的时候会给提示 ，如下：
   //rm: 是否删除 "important.txt"? 
   ```

3. ```c++
   //如果希望在删除文件时不接收任何确认提示，并强制删除文件，可以使用 -f 参数：
   rm -f 文件名
   ```

### 使用==vim==

- 按i键插入，按ESC键并保存更改。输入:w保存更改
- 按 : 键，再输入q!(强制退出)或者wq。



### 使用ls

```c++
ls 	-l
```

每一行前面会出现10个字符

第一个字符，d可能指的是文件夹

后面每三个字符一个组，第一组是我跟人拥有，第二组是同组的人拥有，（同组的啥意思我现在也不是很懂），第三组是其他的人的权限

==三个字符的意思==:"r": read,"w":write,"x":execute;

x的意思是允许你可以访问这个文件夹
