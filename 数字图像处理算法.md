#### 链码

> 链码是对边界点的一种编码表示方法，其基本思想是利用一系列具有特定长度和方向的相连的直线段来表示目标的边界

一般有两种链码：**4方向链码和8方向链码**。我们可以从任意点开始，然后通过方向码（0123这些）表示如何从我们选的任意点开始遍历轮廓回到起点。

显然，这样的话，起点不同，每次链码就不一样了，所以就出现了链码归一化，就是把所有起点对应的链码罗列出来，选择字典序最小的链码。

这样的链码对旋转的**抗干扰性**还是太差了，所以就有了旋转归一化。

计算步骤就是在，**起点归一化后，**我们对归一化后的链码计算**一阶差分链码**（计算方法：计算当前位的链码，通过计算下一位链码与当前位链码的差值然后取模方向数（4,8），-1取模4是-3的意思）。

#### 骨架

表示一个平面区域结构形状的一种重要方法是把它削减成图形。这种削减可以通过细化（也称为抽骨架）算法，获取区域的骨架来实现

对于满足以下四个条件的边界点打标记准备删除：

(a) **2** ≤ **N(p1)** ≤ **6**（N(p1) =p2+p3+…+p9，是点p1邻域中1的个数）

N（p1）就是这个像素点周围像素为1的个数

(b) **S(p1) = 1** (S(p1)是按p2,p3,…,p9顺序，0-1转换的个数)

比如从p2开始，0 0 1 1 1 0 0 这个就是2次了。

(c) **p2 \* p4 \* p6 = 0**（p2 、p4 、p6 至少有一个0）

(d) **p4 \* p6 \* p8 = 0**（p4 、p6 、p8 至少有一个0）
