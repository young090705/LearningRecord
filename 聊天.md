# T酱的聊天教学
## 五种尬聊方式
1. 滔滔不绝式
意思就是**喋喋不休谈论只有我自己感兴趣的事情**
2. 各说各话式
两个人不管彼此在聊什么，只说自己的话，不去回应对方
3. 躲避关注式
当人们认为谈话对象比自己优秀时，会尝试躲避所有的注意力，有可能会说不出话，或者大脑短路词不达意，这样的话表现出来的症状就是毫无理由的预设，对面的人一定不会喜欢自己，导致浪费了进一步互相了解彼此的时间
4. 调查式
想着把话题聚焦到他人身上可能是出自于好心，但是连珠炮式的提问会让人觉得有压力，或者让人觉得我们没把重心放在交流这件事上，不重视交流这件事
5. 挑剔式
老是否认别人提出的观点。
比如别人说电影好看，然后你就说电影一般

上面的问题在于**沟通中有一个人只说不听或者只听不说**，会制造单方面的无效沟通。
沟通的质量取决于你们双方在聊天中，彼此回应，彼此肯定，彼此理解增量。因此，沟通的本质是双向的

## 评论区B友的谈话
比如说 
A：我今天买了杯奶茶
B：我也喜欢喝奶茶嗷（这样 一场问答就结束了）  
B: 哇，听起来像是给自己安排了一个小奖励，今天有什么开心事吗？
这样就关注到了分享背后的情绪和故事和延伸的可能性
>任何分享背后，都有情绪、故事和延伸的可能性。

### dick的讲解
实战升级：让话题自然流动的进阶技巧
1. 分享关联 + 抛回问题
这是最自然的方式：分享**一个与对方话题相关的个人小故事或感受**，然后把话语权抛回去。
A：我上周末去爬山了。
B：“我也超爱爬山！上个月我去了XX山，山顶的云雾简直像仙境。你这次去的哪座山？感觉怎么样？”
（公式：共情 + 个人简要经历 + 开放性问题）

2. 关注对方话语中的“钩子”
对方话里总有可以抓取的细节，比如“终于”、“第一次”、“差点”。
A：“我昨天终于把那个难缠的项目做完了！”
B：“‘**终于**’这个词听起来就很有故事！过程中最让你头大的是什么？”
（钩子：“终于”——背后是拖延、困难或坚持）

3. 从“事情”聊到“人”与“感受”
这是让对话产生深度连接的关键。从外部事件，转向内在体验。
A：“我刚看完《奥本海默》。”
普通回应：“好看吗？我也看了。”（话题可能终结）
深度回应：“我看了之后心情特别复杂。电影里哪个角色或场景让你印象最深？为什么？”
（**从评价电影，转向探讨对方的思考和价值观**）
4. 适时自我暴露，创造安全感
当对方分享一些稍显私人或需要勇气的话题时，**通过适度的自我暴露来表达理解，能极大增进亲密感。**
A：“最近工作压力好大，有点失眠。”
B：“我完全理解，我项目截止前也经常这样。你一般会做点什么让自己放松下来？”

劝各位，聊天从来都不是靠学理论就会的，你需要实战练习和培养共情力，还要改掉自己讨人厌的坏毛病。我就是曾经一个社恐，到大学才敢交朋友的那种，现在大学毕业了，我可以和一个没认识的人聊上一个下午一个晚上都不会停。我觉得跟**别人交流首先要礼貌，然后是对方是否对你的话题感兴趣，还有判断对方是否这个点想聊天**，如果这三个前提都没有，基本也不可能聊的下去



# 本子在隔壁的人与人关系的教学
人与人之间的关系可以简单分为弱连接和强连接
## 两种连接的价值
### 弱连接
代表着创新和机会，找工作，**找爱情**，找合作伙伴，这个就可以帮到我
大脑里需要新的人和新的想法
#### 怎么样去建立多的弱连接呢？
- 大部分的人际交流都是轻信息的搬运工，就是聊聊八卦，烂梗，段子
还有就是聊聊自己身上的小故事，我觉得我可以讲讲我名字的故事
所以记得在适当的时候**收集归纳并提炼这些轻信息**，就是**理解并传递** 轻信息
- 多结识在社交中可能承当桥梁的人，比如社牛，群主，猎头，活动组织者
还有就是不要太在意弱连接所带来的情感以及他们的想法
- 临近性 我们喜欢身边的人
- 重复性 我们喜欢重复接触到的人
- 互惠性 我们喜欢那些喜欢我们的人
- 相似性 喜欢那些与我们相似的人
- 得到性 喜欢那些我们得不到得人
多冒泡 多找共同点 喜欢别人 保持微笑 做一个正能量的正常人，对于大部分弱连接来说已经足够了
### 强连接
有强连接的人会更幸福更长寿，除此之外，我们的性格都是受身边最亲近的人的影响，一个名人所说：“你是与你相处时间最多的五个人所带来的平均值”
#### 怎么样去建立强连接呢？
- 如果一个人不愿意表达出自己的脆弱，还觉得身边人过分矫情，那么基本与建立强连接无缘了
- **问出高质量的有灵魂的问题**，根据生活情况，提出真正触及心灵的问题 有个国外的节目《Hot ONES》
- 

**结论**:不需要用熟人提升自己，也不需要用不熟的人证明自己
原因是熟人带给你的认知过于重复，新结交的人才能带来更多新的知识和事物
